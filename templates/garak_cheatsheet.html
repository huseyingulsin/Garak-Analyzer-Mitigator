📜 Common Garak Commands

Use these commands to run probes, test model behaviors, or evaluate robustness. You can combine models, probes, detectors, and more!

🚀 Basic Commands

  • garak run — Runs the default configuration
  • garak run -m openai:gpt-3.5-turbo — Use a specific model
  • garak run -m ollama:llama3 -p promptinject — Test a local Ollama model

🧪 Probe & Generation Control

  • garak run -p promptinject — Run Prompt Injection probe
  • garak run -p jailbreak -g 20 — Run 20 generations with the jailbreak probe
  • garak run -p regex,grammar — Run multiple probes
  • garak list probes — List all available probes

🧠 Model Types

  • -m openai:gpt-4 — OpenAI API model
  • -m huggingface:tiiuae/falcon-7b — Hugging Face model
  • -m ollama:gemma — Local model via Ollama

🔍 Detectors

  • -d auto — Use automatic detector (recommended)
  • garak list detectors — View all detectors

📝 Output & Reporting

  • --report jsonl — Save results in JSONL format
  • --report html — Generate HTML report
  • --report_dir results/ — Save reports in a custom folder

⚙️ Advanced

  • --generations 50 — Set number of generations
  • --interactive — Step through attempts manually
  • --deprefix — Strip prefixes before scoring

❓ Help

  • garak --help — See full help and options
  • garak run --help — Detailed options for `run`

Pro tip: Combine everything for full control:


garak run -m openai:gpt-4 -p promptinject,jailbreak -d auto -g 10 --report jsonl
