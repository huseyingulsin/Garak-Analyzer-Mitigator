ü§ñ Run Garak on Local LLMs via Ollama

Garak can be used to test models running locally on your machine through Ollama. This is perfect for privacy, speed, or experimenting with open-source models.

üß± Step-by-Step Setup

  1. Install Ollama
    Mac: 
    brew install ollama
    Or visit: ollama.com/download
  2. Pull a model (e.g., Gemma or LLaMA3) 
    ollama pull gemma
ollama pull llama3
  3. Start the model (optional, Garak can auto-trigger) 
    ollama run gemma
  4. Run Garak with Ollama 
    garak run -m ollama:gemma -p promptinject -g 5 --report jsonl
